Location:

Job Summary:
As an entry-level Software Engineer in the Application Performance team, you will be responsible for analyzing, optimizing, and enhancing the performance of our software applications. You will work closely with cross-functional teams to identify bottlenecks, develop solutions, and ensure our applications run efficiently at scale.
Key Responsibilities:
Performance Analysis:
Monitor and analyze application performance metrics to identify potential issues and areas for improvement.
Optimization:
Implement performance enhancements by optimizing code, algorithms, and system configurations.
Troubleshooting:
Debug performance issues, resolve bottlenecks, and collaborate with development teams to implement fixes.
Testing:
Develop and execute performance test cases to validate the effectiveness of optimizations.
Code Review:
Participate in code reviews to ensure that performance best practices are followed during the development process.
Documentation:
Maintain comprehensive documentation of performance metrics, findings, and optimizations.
Collaboration:
Work with cross-functional teams, including software developers, QA engineers, and system administrators, to ensure optimal application performance.
